## MateriApps LIVE!のインストール

- はじめてお手軽に材料科学計算を行うのに便利なツール。東大物性研ほかが提供。
- 仮想デスクトップソフトウェア(VirtualBox)上に、計算環境がすでに整ったLinux (Debian)をブートする。
- 計算環境構築で最初の難関、ソフトウェアのコンパイルをせずにすぐ計算を体験できる。
